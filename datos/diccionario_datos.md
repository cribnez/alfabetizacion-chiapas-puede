# 📖 Diccionario de Datos

Este documento describe las variables contenidas en el archivo `dataset_chiapas_puede_2025.csv`. Los datos han sido **anonimizados** para cumplir con los protocolos de ética en investigación social.

---

## Descripción de las Columnas

| Columna | Tipo de Dato | Descripción |
| :--- | :--- | :--- |
| **ID_Educando** | Alfanumérico | Identificador único anónimo (EDU-01, EDU-02, etc.). |
| **Género** | Categórico | Identidad de género del participante (Femenino / Masculino). |
| **Estatus_Final** | Categórico | Estado del educando al cierre del ciclo (Acreditado / Deserción). |
| **Certificado** | Booleano | Indica si obtuvo el certificado oficial (Sí / No). |
| **Observaciones** | Texto | Notas etnográficas sobre barreras logísticas o facilitadores. |

---

## Notas Metodológicas
* **N = 19**: El tamaño de la muestra corresponde a la micro-región Tuxtla Gutiérrez Oriente.
* **Fuente**: Datos sistematizados a partir de la Plataforma SASA y diarios de campo del investigador.
